// ============================================================
// WinCC 画面设计说明
// 冷库温度控制系统 - WinCC 上位机
// ============================================================
// 共5个画面, 覆盖监控/操作/报警/趋势全功能
// ============================================================


// ============================================================
// 画面1: 总览画面 (Overview.pdl)
// ============================================================
// 功能: 一屏显示9个冷库的整体运行状态
//
// 布局:
// ┌─────────────────────────────────────────────────────┐
// │              冷库温度控制系统 - 总览                  │
// │  [全局状态栏] 运行:XX台  故障:XX台  报警库:XX个       │
// ├──────────┬──────────┬──────────┐                   │
// │  冷库1   │  冷库2   │  冷库3   │                   │
// │ -18.5degC│ -19.2degC│ -17.8degC│                   │
// │ 运行:8台  │ 运行:10台 │ 运行:6台  │                   │
// │ 故障:0   │ 故障:0   │ 故障:1   │                   │
// │ ●正常    │ ●正常    │ ●报警    │                   │
// ├──────────┼──────────┼──────────┤                   │
// │  冷库4   │  冷库5   │  冷库6   │                   │
// │  ...     │  ...     │  ...     │                   │
// ├──────────┼──────────┼──────────┤                   │
// │  冷库7   │  冷库8   │  冷库9   │                   │
// │  ...     │  ...     │  ...     │                   │
// ├──────────┴──────────┴──────────┤                   │
// │ [报警] [趋势] [参数] [退出]      │                   │
// └─────────────────────────────────────────────────────┘
//
// 元素说明:
//   - 每个冷库卡片: 3行文本 + 状态指示灯
//   - 第1行: 冷库名称 (如"冷库1")
//   - 第2行: 实际温度 (大字号显示, 单位degC)
//   - 第3行: 运行台数/故障台数
//   - 状态指示灯: 绿色=正常, 黄色=预警, 红色=报警, 灰色=通信故障
//   - 点击任一卡片 → 跳转到该库详情画面
//
// 颜色方案:
//   - 温度正常: 绿色背景
//   - 温度超限: 红色背景 (闪烁)
//   - 有故障: 黄色边框
//   - 通信故障: 灰色背景


// ============================================================
// 画面2: 单库详情画面 (RoomDetail.pdl)
// ============================================================
// 功能: 显示单个冷库的详细运行状态和操作面板
//
// 布局:
// ┌─────────────────────────────────────────────────────┐
// │  冷库1 - 详情                    [返回总览]          │
// ├──────────────────────┬──────────────────────────────┤
// │  温度显示             │  温度趋势图 (近2小时)        │
// │  ┌───────────────┐   │  ┌────────────────────────┐ │
// │  │   -18.5 degC  │   │  │  温度曲线              │ │
// │  │   (实际温度)   │   │  │  ----上限线            │ │
// │  └───────────────┘   │  │  ----下限线            │ │
// │  传感器1: -18.3 degC │  └────────────────────────┘ │
// │  传感器2: -18.7 degC │                              │
// │  蒸发器:   -5.2 degC │  参数设置                   │
// │                      │  启动温度: [-16.0] degC     │
// │  设定值              │  停止温度: [-20.0] degC     │
// │  上限: -16.0 degC    │  报警温度: [-10.0] degC     │
// │  下限: -20.0 degC    │                              │
// │  报警: -10.0 degC    │  操作按钮                   │
// │                      │  [系统使能 ON/OFF]          │
// ├──────────────────────┤  [手动化霜]                 │
// │  压缩机状态 (10台)    │  [急停] (红色, 带确认)      │
// │  ① ② ③ ④ ⑤          │  [报警确认]                 │
// │  ⑥ ⑦ ⑧ ⑨ ⑩          │                              │
// │  绿=运行 红=故障      │                              │
// │  灰=停机 黄=隔离      │                              │
// │                      │                              │
// │  运行: 8台            │                              │
// │  故障: 0台            │                              │
// │  可用: 10台           │                              │
// └──────────────────────┴──────────────────────────────┘
//
// 压缩机状态指示灯:
//   - 圆形指示灯 × 10, 编号1-10
//   - 绿色 = 运行中 (wCompRunning.Bit[n-1] = 1)
//   - 红色 = 故障 (wCompFault.Bit[n-1] = 1, 闪烁)
//   - 灰色 = 停止 (运行=0, 故障=0)
//   - 黄色 = 隔离 (手动旁路)
//   - 点击指示灯 → 弹出该台压缩机详情 (运行时间/启动次数/手动操作)
//
// 温度趋势图:
//   - X轴: 时间 (近2小时, 可切换为24小时/7天)
//   - Y轴: 温度 (degC)
//   - 曲线: 实际温度 (Room1_Temp_Actual)
//   - 水平线: 上限/下限/报警值 (不同颜色)
//   - 支持缩放和拖拽


// ============================================================
// 画面3: 报警画面 (Alarm.pdl)
// ============================================================
// 功能: 显示当前报警和历史报警记录
//
// 布局:
// ┌─────────────────────────────────────────────────────┐
// │  报警记录                       [返回总览]          │
// ├─────────────────────────────────────────────────────┤
// │  [当前报警] [历史报警]                              │
// ├─────────────────────────────────────────────────────┤
// │ 时间        | 冷库  | 报警内容      | 状态  | 优先级│
// │-------------|-------|--------------|-------|-------│
// │ 14:23:05    | 冷库3 | 压缩机7故障   | 未确认│ 中    │
// │ 14:20:12    | 冷库1 | 超温报警      | 已确认│ 高    │
// │ 14:15:33    | 冷库5 | 通信故障      | 已恢复│ 高    │
// │ ...         | ...   | ...          | ...   | ...   │
// ├─────────────────────────────────────────────────────┤
// │  [确认所有] [确认选中] [导出] [筛选▼]               │
// └─────────────────────────────────────────────────────┘
//
// 使用WinCC Alarm Control控件:
//   - 当前报警: 显示未恢复的报警
//   - 历史报警: 显示所有报警记录 (可按时间/冷库/优先级筛选)
//   - 报警状态: 未确认(红色) / 已确认(黄色) / 已恢复(绿色)
//   - 支持: 确认/筛选/排序/导出


// ============================================================
// 画面4: 参数设置画面 (Settings.pdl)
// ============================================================
// 功能: 集中设置9个冷库的温度参数和系统参数
//
// 布局:
// ┌─────────────────────────────────────────────────────┐
// │  参数设置                       [返回总览]          │
// ├──────┬──────────┬──────────┬──────────┬───────────┤
// │ 冷库 │ 启动温度  │ 停止温度  │ 报警温度  │ 化霜周期  │
// │      │ (degC)   │ (degC)   │ (degC)   │ (小时)    │
// ├──────┼──────────┼──────────┼──────────┼───────────┤
// │ 库1  │ [-16.0]  │ [-20.0]  │ [-10.0]  │ [6]       │
// │ 库2  │ [-16.0]  │ [-20.0]  │ [-10.0]  │ [6]       │
// │ ...  │ ...      │ ...      │ ...      │ ...       │
// │ 库9  │ [-16.0]  │ [-20.0]  │ [-10.0]  │ [6]       │
// ├──────┴──────────┴──────────┴──────────┴───────────┤
// │  [保存] [恢复默认] [批量设置]                       │
// └─────────────────────────────────────────────────────┘
//
// 安全措施:
//   - 参数修改需要操作员权限 (Password Level 2)
//   - 保存时弹出确认对话框
//   - 参数范围校验 (如: HighLimit > LowLimit)
//   - 批量设置: 可一次设置所有库相同参数

// │  ┌────────────────────────────────────────────────┐ │
// │  │                                                │ │
// │  │  温度                                          │ │
// │  │  -10 ──────────────────────────────────────    │ │
// │  │  -15 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─    │ │
// │  │  -18 ●●●●●●●●●●●●●●●●●●●●●●●●●●●●●           │ │
// │  │  -20 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─    │ │
// │  │  -25 ──────────────────────────────────────    │ │
// │  │      00:00  06:00  12:00  18:00  24:00  时间   │ │
// │  └────────────────────────────────────────────────┘ │
// │                                                     │
// │  图例: ─库1 ─库2 ─库3 ─库4 ─库5 ─库6 ─库7 ─库8 ─库9│
// └─────────────────────────────────────────────────────┘
//
// 使用WinCC Online Trend Control控件:
//   - 多曲线对比 (最多9条温度曲线, 不同颜色)
//   - 时间范围可调 (2小时/24小时/7天/30天)
//   - 支持缩放/拖拽/游标读取数值

// ============================================================
// 用户权限管理
// ============================================================
// 权限级别:
//   Level 0 - 观察 (默认): 只能查看画面, 不能操作
//   Level 1 - 操作员: 可启停系统/手动化霜/报警确认
//   Level 2 - 工程师: 可修改温度参数/化霜周期等设定值
//   Level 3 - 管理员: 可修改所有参数/用户管理/导出数据
//
// 登录方式: WinCC用户管理, 用户名+密码
// 自动登出: 30分钟无操作自动退出到Level 0
